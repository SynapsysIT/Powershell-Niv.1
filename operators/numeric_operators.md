# Comparer des nombres

| Commande                | Description         |
| ----------------------- | ------------------- |
| `2 -eq 2`  | Egal                |
| `2 -ne 4`  | N'est pas égal      |
| `5 -gt 2`  | Plus grand que      |
| `5 -ge 5`  | Plus grand ou égale |
| `5 -lt 10` | Plus petit que      |
| `5 -lt 5`  | Plus petit ou égale |
